My Projects

AR Wall

I was the major contributor to this project during my research assistantship at IMAGINE Lab, Center of GIS, Georgia Tech. This project is intended to build an augmented reality application that will assist civil engineering students and also building managers to have a direct view of in-wall structure. I built the 3D model of in-wall structure in 3ds Max and imported into Unity 3D to implement augmented reality function using Qualcomm Vuforia package. I also built a simple user interface through C# scripts. The target platform of this application is Android and was successfully built and run on Nexus. However, since the application was built on school device, the video will show a test running of a previous version of the application in Unity.

Technologies used:

  • 3ds Max
  • Unity 3D
  • Qualcomm Vuforia
  • C#
